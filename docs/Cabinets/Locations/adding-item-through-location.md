# Add Item to Location Through Location Module

Adding an item to a location allows users to **assign inventory to a cabinet bin**, set minimum and maximum quantities, and optionally enter starting inventory counts. This process is completed through the **Cabinet Module**

**Also known as:** *Assign Item to Location*, *Create Bin Location*, *Add Inventory Location*

---

## How to Add an Item to a Location

### 1. Open Cabinet Module

1. From your dashboard, click **Cabinet Module**.
2. This opens the list of available cabinets for your site.

---

### 2. Select a Cabinet

1. In **Cabinet List**, choose the cabinet where the item will be stored.
2. Click into that cabinet to view available bin locations. 

---

### 3. Find an Open Location

You can locate an empty bin in two ways:

- **Scroll through available pages** and click **Select** on an open location


---

### 4. Select the Item

1. On the **Item** line, click **Select**.
2. In the search box, type the item name.
3. When your item appears, click **Select**.

If the item does not already exist:

- Click **Create Item**, or
- Click the **(+) Add** button to create a new item first

---

### 5. Configure the Bin

After selecting the item:

- Enter the **Minimum Quantity**
- Enter the **Maximum Quantity**
- Enable **Location Expansion** if multiple items share the same bin
- Check **Restock** if the item should be available for reorder

---

### 6. Add Starting Inventory (Optional)

If inventory is being stocked immediately:

1. Click the linked number in **On Hand Qty**
2. Enter the correct quantity
3. Click **OK** to save the count and return to the bin page

---

### 7. Update the Location

Before saving:

- Verify the item is correct
- Confirm **Min / Max** quantities
- Confirm **On Hand Qty** if inventory was added
- Review any additional settings

Click **Update Location** to complete setup.

---

## Completed Setup

Once complete:

- The item is assigned to the cabinet location
- Inventory count is stored in the system
- The location is now active for checkout and restocking
