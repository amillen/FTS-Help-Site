# Receive Gage After Calibration

Receiving a gage after calibration returns it to **active inventory** and records calibration results, documentation, and status. This process ensures gages are properly tracked, compliant, and ready for use.

**Also known as:** *Receive Calibration*, *Return Gage to Inventory*

---

## How to Receive a Gage After Calibration

### 1. Prepare Documentation

Before receiving the gage in the system:

- Save any **calibration certificates** or documentation provided by the calibrator
- Ensure files are available in **PDF format** for upload

---

### 2. Attach Calibration Certification (PO Level)

When the gage returns:

1. Locate the calibration order
2. In the **Attachment** area:
   - Click **Choose File**
   - Select the calibration certificate PDF
3. The file uploads using the existing file name

> **Note:** If you receive an error indicating the order is already received, the process is complete and no further changes can be made.

---

### 3. Attach Certification at the Line Level (Optional)

If certification applies to a specific gage line:

1. Expand the gage line
2. Click **Choose File**
3. Select the appropriate certification document
4. Click **Save** to store the attachment

This allows detailed tracking at the individual gage level.

---

### 4. Receive the Gage into the System

1. Navigate to **Out For Calibration**
2. Select the gage being returned
3. Click to receive the gage

A **Receive Gage After Calibration** dialog box will appear.

---

### 5. Enter Calibration Details

In the dialog:

- Enter the **Date Calibrated**
- Add notes from the **Inspection Report Findings**
- Indicate whether the gage **exceeded tolerance**

---

### 6. Set Gage Status

Choose the appropriate status to reflect how the gage returned:

- **Active** – Calibrated and ready to use
- **Broken** – Damaged and unusable
- **Inactive** – Out of use
- **Lost** – Tracked but not physically present
- **Needs Calibration** – Returned but not calibrated correctly
- **Removed From Service** – Not currently in use

---

### 7. Finalize and Save

- Review all entered data
- Click **OK** to complete the receive process
- Click **Save** to ensure all attachments and data are stored

Once saved, the calibration order status is updated to **Received**.

---

## Important Notes

- Files **must be saved** or they will not be available for future viewing
- Completed orders **cannot be modified**
- Proper status selection is critical for compliance and inventory accuracy
