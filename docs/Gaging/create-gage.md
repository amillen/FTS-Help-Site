# Creating a Gage

Creating a gage allows users to **add calibrated measurement tools** to the system, define calibration requirements, and track individual gage usage and status.

**Also known as:** *Create Gage*, *Add New Gage*, *Gage Setup*

---

## How to Create a New Gage

### 1. Open the Items Module

1. Log in to your Cloud dashboard.
2. Click on the **Items** module.

You will see the **Item List**, which displays all tools currently in the machine.  
You may select an existing item to edit or create a new one.

---

### 2. Create a New Item

To create a new item:

1. Click **Create New Item**, or  
2. Click the **plus (+) icon** at the bottom of the screen.
3. Fill in the item details [Creating Item](xref:items.add).  
4. Click **Create Item** to create the item. 


---

### 4. Configure the Item as a Gage

1. In the **Type** dropdown, select **Gage**.
2. In the **Calibrator** field, enter the supplier responsible for calibrations.
3. Click **Update Item**.

---

## Configure Gage Specifications

### 5. Update Gaging Information

1. Select the **Gaging** tab.
2. Enter the following information:
   - Calibration price
   - Inspection steps (if required)
3. Confirm the **calibration supplier**.

Optional tracking features:
- **Date Tracking** – restricts gage use by number of days
- **Gage Usage Tracking** – restricts use by number of cycles

Click **Update Item** to save your changes.

---

## Add Individual Gage Data

### 6. Add Gage-Specific Information

1. Click the **blue plus (+) icon** above **Update Item**.
2. Enter the required gage details:
   - Serial Number
   - Manufacturing Number
   - Date of Last Calibration
   - Gage Status
   - Notes (optional)
3. Click **OK**.

---

## Gage Added Successfully

The new gage will now appear at the bottom of the item record, identified by its **serial number**, along with all associated calibration and usage information.
