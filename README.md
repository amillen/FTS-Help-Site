# <Project Name> Docs (DocFX)

Documentation site built with [DocFX].  
Contributions welcome via pull requests!

## Build locally

1. Install DocFX (Windows):
   - Via Chocolatey: `choco install docfx -y`
2. Build:
   - `docfx docfx.json`
3. Serve locally:
   - `docfx serve _site`
